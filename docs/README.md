## ‚ùì Troubleshooting

### WhatsApp buttons not appearing?
- Check browser console for errors
- Verify `whatsapp-integration.js` is loaded
- Ensure script tag is before `</body>`

### Carousel not responsive?
- Clear browser cache
- Verify carousel CSS is updated
- Check mobile breakpoints in CSS

### Images loading slowly?
- Verify `loading="lazy"` is applied
- Check image file sizes (should be optimized)
- Test on different network speeds

### Hero backgrounds not showing?
- Check image paths are correct
- Verify CSS variable syntax: `style="--bg-image: url('...')"`
- Inspect element to see if CSS is applied
